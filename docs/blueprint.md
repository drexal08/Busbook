# **App Name**: EZ Bus

## Core Features:

- Search Routes: Allow users to search for available bus routes based on origin, destination, and date.
- Route Display: Display available bus routes with details like departure time, arrival time, duration, price, and available seats.
- Seat Selection: Enable users to select their preferred seats from a seat map.
- Booking Management: Allow users to manage and view their booking details.

## Style Guidelines:

- Primary color: Dark indigo (#3F51B5) for trust and reliability.
- Background color: Light gray (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Teal (#009688) to highlight interactive elements and call-to-actions.
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, machined, objective, neutral look; suitable for both headlines and body text
- Use simple, consistent icons to represent different aspects of the journey.
- Implement a clean and intuitive layout, with clear hierarchy and spacing, to enhance usability and reduce visual clutter.
- Incorporate subtle transitions and animations to enhance the user experience, such as loading animations and smooth transitions between pages.
